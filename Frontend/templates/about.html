This is a simple get-and-graph tool that draws on word data from the French national library's online archive of newspapers, mostly. It is the author's first foray at API integration and website design, so any and all suggestions are welcome!



Example usages:

Search term: brazza
Newspapers: all
Year range: 1860-1900
Restrict year range: true (box checked)

This would search through all newspapers on Gallica's website for the term "brazza".
